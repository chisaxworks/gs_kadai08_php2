## ②課題内容（どんな作品か）
- 契約管理ツール（SQL版）:前回作成したツールをデータベース化しました
- 変更点
  - 入力欄をデフォルトで非表示にしました ★登録時は右上の円をクリックします

  - カード型の部分は、利用サービス名とOGPの情報を表示させるようにしました

  - 一応OGP画像はキャッシュに保存するようにはしました（少しだけ軽くなったかも）
